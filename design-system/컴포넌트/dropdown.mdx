옵션 목록을 펼침/접힘으로 표시하는 복합 선택 컴포넌트. Title / Content / Slot 서브 컴포넌트로 구성된 compound 패턴을 따른다.

<Note>

상세 스펙(Anatomy, Props, 토큰 매핑)은 01 Project/SAZO/SAZO - 디자인 시스템 점검/디자인 시스템/03 AI 컨텍스트/context/components/dropdown-spec을 참조한다.

</Note>

## 언제 사용하는가
- 선택 옵션이 **5개 이상** 일 때 Radio/Chip 대신 사용한다.
- 제한된 화면 공간에서 옵션 목록을 숨겼다가 보여줘야 할 때.
- Slot에 이미지 + 텍스트 복합 콘텐츠를 넣어 풍부한 옵션을 표시할 때.

## Do / Don't

<Check>

**Do**

- `isOpen`과 `onOpenChange`로 상태를 제어하여 예측 가능한 동작 보장 — *Radio 또는 Chip이 적합*
- Title에 placeholder 텍스트를 넣어 미선택 상태 명시 — *UX 복잡도 급증*
- Slot에 이미지 + 텍스트를 조합하여 풍부한 옵션 표시 — *선택 가능 옵션이 너무 적으면 혼란*
- Content의 maxHeight(360px) 초과 예상 시 검색 기능 함께 고려 — *원하는 옵션 찾기 어려움*

</Check>

<Warning>

**Don't**

- 옵션이 **2~3개** 뿐인 경우 사용
- 중첩된 Dropdown 사용
- Slot의 `disabled`를 과도하게 사용
- 검색 없이 긴 목록만 스크롤로 제공

</Warning>

## 패턴: Compound Component
Dropdown은 `Dropdown.Title`, `Dropdown.Content`, `Dropdown.Slot`을 조합하는 compound pattern이다.

```jsx
<Dropdown isOpen={open} onOpenChange={setOpen}>
  <Dropdown.Title title={selected} isSelected={!!selected} onClick={toggle} />
  <Dropdown.Content isOpen={open}>
    <Dropdown.Slot onClick={() => select('A')}>Option A</Dropdown.Slot>
    <Dropdown.Slot onClick={() => select('B')}>Option B</Dropdown.Slot>
  </Dropdown.Content>
</Dropdown>
```

외부 클릭 시 자동으로 `onOpenChange(false)`가 호출된다.
