### 언제 사용하는가
상품 정보(이미지, 제목, 가격, 할인율, 평점 등)를 **구조화된 카드 레이아웃** 으로 표시할 때 사용한다. SAZO의 핵심 상품 표시 컴포넌트다.

### 대안 분기표
| 상황 | ItemCard 대신 사용할 컴포넌트 | 이유 |
|------|--------------------------|------|
| 옵션 선택 카드 (배송 방법, 크기 등) | TileSelector | ItemCard는 정보 표시용이며, 선택 상태 관리를 내장하지 않음 |
| 정보 안내/배너 | InfoPanel | 상품이 아닌 안내 메시지는 InfoPanel이 적합 |
| 빈 목록 표시 | EmptyState | 상품이 없을 때는 빈 상태 전용 컴포넌트 사용 |
| 순수 이미지 갤러리 | 커스텀 이미지 그리드 | ItemCard는 텍스트 상세(제목, 가격)가 필수 구조 |

### 선택 가이드
### View 선택
| View | 레이아웃 | 사용 맥락 | 배치 위치 |
|------|---------|-----------|-----------|
| `grid` | 세로 카드 (이미지 위, 텍스트 아래) | 상품 목록 그리드, 카테고리 브라우징, 홈 추천 | 2~3열 그리드 (모바일 2열, PC 3~4열) |
| `list` | 가로 카드 (이미지 좌, 텍스트 우) | 검색 결과, 주문 내역, 관련 상품 | 단일 열 리스트, 수평 스크롤 |

### 크기(Size) 선택
| Size | 가격 폰트 크기 | 사용 맥락 | 배치 위치 |
|------|-------------|-----------|-----------|
| `xs` | Grid 16px / List 18px | 소형 카드, 보조 영역 | 최근 본 상품, 사이드바 추천, 풋터 영역 |
| `sm` | Grid 18px / List 18px | 기본 상품 카드 (default) | 메인 상품 목록, 검색 결과, 카테고리 |
| `xl` | Grid 18px / List 18px | 대형 카드, 강조 상품 | 히어로 섹션, 프로모션 상품, 상세 추천 |

### 이미지 AddOn 배치
| Position | 용도 | 권장 요소 |
|----------|------|----------|
| `tl` (좌상단) | 순위 표시 | `ItemCardRank` |
| `tr` (우상단) | 북마크/좋아요 | `ItemCardLikeButton` |
| `bl` (좌하단) | 보조 배지 | Badge 또는 커스텀 |
| `br` (우하단) | 보조 액션 | 거의 사용하지 않음 |

AddOn은 **최대 2개** 까지만 사용한다. 4개를 모두 채우면 이미지 가독성이 심각하게 저하된다.

### Rank Badge 색상
| Color | 토큰 | 사용 맥락 |
|-------|------|-----------|
| `default` | `--opacity-black-900` | 일반 랭킹 |
| `brand` | `--surface-brand` | 브랜드 강조 랭킹 |
| `black` | `--text-default` | 다크 랭킹 |
| `purple` | `--text-benefit` | 프로모션/혜택 랭킹 |

### 배치 규칙
| 배치 패턴 | 설명 | 간격 |
|----------|------|------|
| Grid 2열 (모바일) | `grid-cols-2` + gap | 카드 간 gap 8~12px, 행 간 gap 16~20px |
| Grid 3~4열 (PC) | `grid-cols-3` 또는 `grid-cols-4` + gap | 카드 간 gap 12~16px |
| List 단일 열 | 세로 리스트 | 항목 간 gap 12~16px 또는 구분선 |
| 수평 스크롤 | Swiper 또는 overflow-x-auto | 카드 간 gap 8~12px |

- 이미지 모서리: `rounded-xl` (`--radius-xl`, 12px) 적용
- 할인율 텍스트: brand-500 (primitive 직접) 사용 -- 사용자의 시선을 할인 정보로 유도

### Do / Don't
| Do | Don't | 이유 |
|----|-------|------|
| `wrapperAs="a"`로 카드 전체를 클릭 가능하게 설정 | 카드 내부 특정 영역만 클릭 가능하게 설정 | 사용자는 카드 어디를 클릭해도 상품 상세로 이동할 것을 기대 |
| `showOutOfStock` + `outOfStockMessage`로 품절 상태 명확히 표시 | 품절 상품을 일반 상품과 동일하게 표시 | 클릭 후 "품절입니다" 안내는 사용자 기대를 저버림 |
| `discount`와 `price`를 함께 제공하여 할인 컨텍스트 완성 | `discount` 없이 할인된 가격만 표시 | 할인율 없이 가격만 보면 혼란 |
| `extraDetailNode`를 활용하여 추가 정보 표시 | 카드 구조를 임의로 수정하여 커스텀 레이아웃 | 일관된 카드 구조가 깨지면 사용자의 시선 패턴 혼란 |
| AddOn은 `tl` + `tr` 조합(순위 + 좋아요)으로 최대 2개 사용 | 4개 position을 모두 채움 | 이미지 위에 요소가 많으면 상품 이미지 가독성 저하 |
| 같은 목록 내에서 동일한 view와 size 유지 | 같은 목록 내에서 grid와 list를 혼합 | 시각적 일관성이 깨지고 사용자 혼란 유발 |
