## 철학
SAZO의 텍스트 계층은 **5단계 시각적 무게 체계** 로 구축된다. 모든 텍스트는 크기-굵기-색상의 3속성 조합으로 정해지며, 이 조합이 곧 정보의 중요도를 결정한다.

- 크기가 클수록 시각적 무게가 크다 (Display > Title > Subtitle > Body > Caption)
- 같은 크기 내에서 `600` > `500` > `400` 순으로 주목도가 높다
- 색상은 `--text-strong` > `--text-default` > `--text-secondary` > `--text-tertiary` 순이다

### 5단계 계층표
```
 계층       시각적 무게  ████████████████████████████████  <-- 높음
 ──────────────────────────────────────────────────────
 1. Display              ████████████████████████████████
 2. Title                ██████████████████████████
 3. Subtitle             █████████████████████
 4. Body                 ███████████████
 5. Caption              █████████
```

| 계층 | 역할 | 크기 | 굵기 | 색상 토큰 | 사용처 |
| --- | --- | --- | --- | --- | --- |
| **1. Display** | 히어로/랜딩 대제목 | `--font-size-48`~`--font-size-60` | `--fw-bold` | `--text-strong` | 히어로 배너, 이벤트 랜딩 |
| **2. Title** | 페이지/섹션 제목 | `--font-size-20`~`--font-size-36` | `--fw-semibold` | `--text-strong` | SectionTitle, FeaturedHeading |
| **3. Subtitle** | 항목 제목, 강조 정보 | `--font-size-16`~`--font-size-18` | `--fw-medium`~`--fw-semibold` | `--text-default` | 카드 상품명, EmptyState 제목, 메뉴명 |
| **4. Body** | 설명, 본문 | `--font-size-12`~`--font-size-18` | `--fw-normal` | `--text-secondary` | 옵션 텍스트, 모달 설명, FAQ 답변 |
| **5. Caption** | 보조/메타 정보 | `--font-size-12`~`--font-size-14` | `--fw-normal`~`--fw-medium` | `--text-tertiary`~`--text-secondary` | 날짜, 배지, 힌트, 만료일 |

## 계층별 토큰 상세

<Tabs>

<Tab title="계층 1: Display">

| 속성     | Semantic Token            | Primitive         |
| ------ | ------------------------- | ----------------- |
| 히어로    | `--type-display-hero`     | `--font-size-60` / `--line-height-60`    |
| 히어로 서브 | `--type-display-hero-sub` | `--font-size-48` / `--line-height-48`    |
| 굵기     | `--fw-display`            | `--fw-bold` (700) |
| 색상     | `--text-strong`           | `--gray-950`      |

사용처: 히어로 배너, 이벤트 랜딩 페이지

#### 간격 패턴
```
[Display: text-display-hero]         ← 화면당 1개만
      mt-5 (20px)
[Body: text-body-large]             ← description
      mb-10 (40px)
[다음 섹션]
```

</Tab>

| 굵기 (라벨) | `--fw-label` | `--fw-medium` (500) |
| 색상 | `--text-default` | `--gray-900` |

```
[배지/태그]            ← 있다면 Subtitle 위
```

## 특수 텍스트 레시피

<Accordion title="가격 표시">

| 용도 | 타이포 토큰 | 색상 토큰 |
|------|-----------|----------|
| 일반 가격 | `text-title-subsection-lg` | `--text-default` |
| 총합계 가격 | `text-title-section-sm` | `--text-default` |
| 항목별 라벨 | `text-body-default` | `--text-secondary` |
| 항목별 금액 | `text-label-large` | `--text-default` |
| 취소선 원래가격 | `--font-size-16`~`--font-size-20` / `--fw-medium` / `line-through` | `--text-disabled` |
| 쿠폰 할인 | `text-title-subsection-sm` | `--text-benefit` |

</Accordion>

<Accordion title="배지 텍스트">

| 변형 | 타이포 토큰 | 색상 토큰 |
|------|-----------|----------|
| 기본 (회색) | `text-caption-small` | `--text-secondary` + `--surface-default` + `--border-default` |
| 쿠폰 (보라) | `text-caption-small` | `--text-benefit-strong` + `--surface-benefit` |
| 카운트 | `text-label-small` | `--text-inverse` + `--surface-brand` |

</Accordion>

<Accordion title="에러 텍스트">

| 용도 | 타이포 토큰 | 색상 토큰 |
|------|-----------|----------|
| 인라인 에러 | `text-caption-default` 또는 `text-caption-small` | `--text-error` |
| 에러 배지 | Badge `color="error" size="xs"` | `--surface-error` + `--text-error` |

</Accordion>

<Accordion title="비활성 텍스트">

모든 비활성 상태는 원래 스타일 유지 + 색상만 `--text-disabled`로 오버라이드한다.

</Accordion>

## FeaturedHeading 레시피
| size | title 토큰                                  | description 토큰                                        | 적합 컨텍스트        |
| ---- | ----------------------------------------- | ----------------------------------------------------- | -------------- |
| `md` | `text-title-section-lg` + `--text-strong` | `text-body-default` + `--text-secondary`              | 모달 안내, 중형 섹션   |
| `lg` | `text-title-section-xl` + `--text-strong` | `text-body-large` + `--text-secondary`                | 가이드, about 페이지 |

공통: 컨테이너 `mb-10`(40px), title-description 간 `mt-5`(20px), `whitespace-pre-line`

## SectionTitle 레시피
공통 스타일: `mb-4`(16px) + `--fw-heading` (600) + `--text-strong`

| size      | Semantic Token             |
| --------- | -------------------------- |
| `2xl`     | `text-title-screen`        |
| `xl`      | `text-title-section-xl`    |
| `lg`      | `text-title-section-lg`    |
| `md` (기본) | `text-title-section-md`    |
| `sm`      | `text-title-section-sm`    |
| `xs`      | `text-title-subsection-lg` |

## Do / Don't

<Check>**Do**</Check>

- Display는 화면당 1개만
- Title/Display 색상은 `--text-strong`
- 카드 내 상품명은 `--fw-medium` (500)
- 본문 설명은 `--text-secondary`
- 가격은 `--fw-semibold` (600) + `--text-default`
- 읽혀야 하는 캡션은 `--text-secondary`, 보조 정보는 `--text-tertiary`
- 배지 텍스트는 `text-label-small` (12px/600) 또는 `text-caption-small` (12px/500)
- Title-콘텐츠 사이는 `mb-4` (16px)

<Warning>**Don't**</Warning>

- 한 화면에 2개 이상의 Display
- Title에 `--text-secondary` 사용
- 카드 상품명에 `--fw-semibold` (600) — Title과 혼동
- 본문에 `--text-default` — Subtitle과 구분 불가
- 가격에 `--fw-normal` (400)
- 모든 캡션을 `--text-tertiary`로 통일
- 배지에 `--fw-normal` (400) — 작은 크기 보상 필요
- `mb-2`나 `mb-8` 사용
