## 철학
SAZO의 인터랙션은 **모든 사용자 행동에 즉각적이고 일관된 시각적 응답** 을 제공한다. 핵심 원칙은 세 가지다.

- 모든 인터랙티브 요소에 호버/포커스 피드백이 있어야 한다
- 색상 전환은 200ms (`--duration-micro`), 레이아웃 전환은 300ms (`--duration-standard`)
- 선택 상태는 배경 + 텍스트를 동시에 변경하여 명확한 대비를 만든다

## 더보기 버튼
홈 등에서 전용 페이지로 이동시키는 역할. 섹션 높이가 높아서 섹션 마지막에서 더보기 버튼에 접근이 어려울 것 같으면 **섹션 하단에도 중복 배치** 한다.

## 모션 속도 가이드
| 범주 | Duration | 용도 |
|------|----------|------|
| `--duration-micro` | 200ms | 호버 색상, 아이콘 회전, 아코디언, 헤더 접기 |
| `--duration-standard` | 300ms | 드롭다운, 카드 리프트, 모달 배경 전환 |
| `--duration-emphasis` | 400ms | 모달 열기/닫기 (비대칭 이징) |
| `--duration-entrance` | 300ms | 콘텐츠 slideIn 등장 |

## 전환 효과 토큰
| 전환 유형 | Semantic Token | Primitive | 적용 대상 |
|-----------|---------------|-----------|-----------|
| 색상 변화 | `--duration-micro` | `transition-colors` (200ms) | 모든 호버 색상 |
| 아이콘 회전 | `--duration-micro` | `transition-transform duration-200` | FAQ, 푸터 화살표 |
| 복합 전환 | `--duration-micro` | `transition-all duration-200 ease-in-out` | 드롭다운 확장 |
| 카드 리프트 | `--duration-standard` | `transition-transform duration-300` | 포워딩 카드 hover |
| 투명도 전환 | `--duration-micro` | `transition-opacity` | 탭 투명도 |

## Do / Don't

<Check>

**Do**

- 모든 인터랙티브 요소에 호버 피드백 제공
- 색상 전환에 `transition-colors` (200ms)
- Primary 버튼 hover는 한 단계 진하게
- 카드 hover는 미세한 배경 변화 (`gray-50`)
- 선택 상태는 배경 + 텍스트 동시 변경
- TileSelector 선택은 `border-2 border-gray-900`
- 입력 포커스는 `--border-strong` 명확한 표시
- sticky `top`은 `--header-height` CSS 변수
- 드롭다운은 route 변경 시 자동 닫기
- 아코디언에 `transition-all duration-200`
- 파괴적 버튼은 `color="error"`
- 취소(좌) + 확인(우) 순서 고정

</Check>

<Warning>

**Don't**

- 클릭 가능한 요소에 호버 피드백 없음
- 즉시 변경 (깜빡임)
- 호버 시 완전히 다른 색상
- 카드에 강한 색상 변화
- 텍스트 색상만 변경 (접근성 문제)
- 배경색으로만 선택 구분
- 포커스 시각적 변화 없음 (접근성 위반)
- `top` 값 하드코딩
- route 변경 후 열린 채 방치
- 즉시 열기/닫기
- 비파괴적 작업에 error 색상
- 좌우 반대 배치

</Warning>
