## 철학
SAZO의 리스트는 **반복되는 정보를 균일한 리듬으로 배열** 한다. 모든 리스트 항목은 `py-4`(16px) 수직 패딩을 공유하며, 좌측 아이콘(24px) + 텍스트 + 우측 네비게이션의 3단 구조를 따른다.

- 항목 간 구분은 `border-b` 또는 `<Divider />`로 시각적 경계를 만든다
- 그룹 간 구분은 `<Divider>` + `pt-4 pb-8` 패턴을 사용한다
- 상태 변화(활성/비활성/읽음)는 `font-weight` + `color` 변화로 표현한다

## Do / Don't

<Check>**Do**</Check>

- 모든 리스트 항목에 `py-4` (16px) 수직 패딩
- 그룹 간 구분에 `<Divider>` + `pt-4 pb-8`
- 좌측 아이콘은 24px + `text-text-default-subtle`
- 활성 메뉴는 600 + `text-text-default` (PC)
- 아코디언 전환은 `transition-all duration-200`
- 읽음 상태는 `text-text-disabled` + 이미지 `opacity-50`

<Warning>**Don't**</Warning>

- 항목마다 다른 수직 패딩
- 간격만으로 그룹 구분
- 아이콘마다 다른 색상
- 배경색으로만 활성 표시
- 즉시 열기/닫기
- 읽음/미읽음 시각적 구분 없음
