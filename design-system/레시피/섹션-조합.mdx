## 철학
SAZO의 페이지 구조는 **예측 가능한 수직 리듬** 으로 구축된다. 핵심은 세 가지 숫자다.

- **48px (`mb-12`)** : 메인 페이지 섹션 간격의 기본 박자
- **16px (`mb-4`)** : SectionTitle 아래의 표준 간격
- **32px (`Divider margin="32"`)** : 결제/설정 페이지의 폼 섹션 구분

모든 페이지는 Page Shell(헤더/콘텐츠/푸터) 위에 섹션들을 쌓는 구조이며, PC와 Mobile은 동일한 리듬을 공유하되 구분자 방식이 다르다(PC: 얇은 Divider, Mobile: 두꺼운 ThickDivider).

## Do / Don't

<Check>

**Do**

- 메인 섹션 간격은 항상 `mb-12` (48px)
- 하단 마진(`mb-*`)으로 섹션 간격 설정
- PC 콘텐츠는 `<Spacer>` 컴포넌트로 감싸기
- Mobile Footer에 `pb-[82px]` 포함
- sticky 요소는 `--header-height` CSS 변수 참조
- PC 결제 페이지는 `bg-gray-50` 배경
- Mobile은 `ThickDivider`로 섹션 구분
- 상품 그리드에 `.card-gird-wrap` 클래스 사용

</Check>

<Warning>

**Don't**

- 섹션마다 다른 간격 (mb-8, mb-16 혼용)
- 상하 패딩(`py-*`)으로 섹션 간격 설정
- 개별 섹션에 `max-w-6xl mx-auto px-12` 직접 작성
- 바텀 도크와 겹치게 방치
- `top` 값 하드코딩
- 기본 흰색 배경
- 마진만으로 섹션 구분 (모바일에서 불분명)
- 개별 gap 값 직접 지정

</Warning>
