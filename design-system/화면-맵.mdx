## 공통 네비게이션
## PC
- **SearchableGnb** : 로고 + 검색 + 찜 + 장바구니(뱃지) + 알림(드롭다운) + 프로필(드롭다운)
- **체크아웃 모드** : 로고 + "주문 확인" 타이틀만 (검색/서브탭 숨김)
- **Footer** : SNS + 회사 소개 + 정책 링크 + 한국 법인 정보 토글
- 숨김: 체크아웃, 인증확인

## Mobile
- **LandingGnb** (홈): 로고 + 검색 + 장바구니(뱃지) + 서브탭
- **BackButtonGnb** (상세 이하): 뒤로가기 + 검색(조건부) + 홈(조건부) + 장바구니(조건부)
- **BottomDockMenu** : 홈 + 알림(뱃지) + 검색 + 찜 + 마이페이지/로그인
  - 숨김: 인증, 카트, 상품 상세, 체크아웃, 탈퇴, 추천, 인증확인
- **Footer** : PC + "문의 시작하기"(채널톡)

## 커머스
| 라우트 | 페이지 | 주요 컴포넌트 |
| --- | --- | --- |
| `/` | 홈 | MainBannerCarousel, SazoPick, SazoRank, SazoSection, ItemCard |
| `/detail/[pid]` | 상품 상세 | ImageCarousel, PriceSection, OptionSelector, Stepper, ShareModal |
| `/cart` | 장바구니 | CartItem, CartSummary, RemoveCartItemModal, CouponModal |
| `/checkout/cart` | 체크아웃 (카트) | AddressSelectModal, PersonalCustomCodeModal, CustomsInfoModal |
| `/checkout/direct` | 체크아웃 (바로구매) | 체크아웃(카트)와 동일 레이아웃, 데이터 소스만 다름 |
| `/order-confirmation/[oid]` | 주문 확인 - 성공 | 주문 상세 링크, 쇼핑 계속하기, 채널톡 |
| `/order-confirmation/[oid]/fail` | 주문 확인 - 실패 | 재결제, 홈, 채널톡 |
| `/order-confirmation/redirect` | 결제 리다이렉트 | 프로세싱 전용 (UI 없음) |
