## 2026년 4월 21일 주간 변경 로그

이번 주 SAZO 디자인 시스템의 주요 변경 사항을 요약합니다.

### 새 기능

- **SAZO 디자인 시스템 출시** — Foundations, Guidelines, Components, Patterns, Content 5개 영역, 총 40개 이상의 페이지로 구성된 디자인 시스템이 정식 공개되었습니다. 디자이너, 기획자, 개발자가 일관된 기준으로 협업할 수 있습니다. [디자인 시스템 개요](https://docs.sazo.kr/design-system/index)
- **인터랙티브 컴포넌트 적용** — 탭(Tabs), 아코디언(Accordion), 노트(Note) 등 인터랙티브 요소가 전체 디자인 시스템 페이지에 적용되어 관련 옵션을 빠르게 비교하고 상세 정보를 접고 펼칠 수 있습니다

### 업데이트

- **디자인 시스템 개요 페이지 탐색 개선** — [디자인 시스템 개요](https://docs.sazo.kr/design-system/index) 페이지의 문서 구조 섹션이 카드 기반 내비게이션으로 변경되었습니다. 원칙, 가이드, 컴포넌트, 레시피, 콘텐츠 5개 영역으로 바로 이동할 수 있어 원하는 정보를 더 빠르게 찾을 수 있습니다
- **Pretendard Variable 서체 적용** — 제목과 본문 서체가 Pretendard Variable로 통일되어 한글·영문 혼용 환경에서 가독성이 향상되었습니다
- **문서 탐색 개선** — 40개 디자인 시스템 페이지의 목차 구조가 정리되어 원하는 정보를 빠르게 찾을 수 있습니다
- **사이트 테마 개선** — Mint 테마 적용과 다크 모드 배경 색상 변경으로 전체적인 레이아웃과 가독성이 개선되었습니다
- **Do/Don't 가이드 시각화** — 기존 테이블 형식의 Do/Don't 가이드가 체크/경고 블록으로 변경되어 모범 사례와 주의사항을 한눈에 파악할 수 있습니다. [버튼 위계](https://docs.sazo.kr/design-system/가이드/버튼-위계), [카드 조합](https://docs.sazo.kr/design-system/레시피/카드-조합) 등 24개 섹션에 적용
- **토큰 아키텍처 플로차트 도입** — [간격](https://docs.sazo.kr/design-system/원칙/간격), [모션](https://docs.sazo.kr/design-system/원칙/모션), [타이포그래피](https://docs.sazo.kr/design-system/원칙/타이포그래피) 페이지의 토큰 구조도가 플로차트로 변경되어 계층 간 관계를 더 직관적으로 파악할 수 있습니다
- **브랜드 색상 통일** — 사이트 전체의 기본 강조 색상이 브랜드 레드(#EB4666)로 통일되어 라이트·다크 모드 모두에서 일관된 브랜드 아이덴티티를 제공합니다
- **파비콘 리디자인** — 파비콘이 새 규격으로 업데이트되어 브라우저 탭과 즐겨찾기에서 더 선명하게 표시됩니다
- **섹션 제목 색상 통일** — 섹션 제목(h2)에 적용되던 브랜드 강조 색상이 제거되어 본문과 일관된 색상 체계로 통일되었습니다. 전체 페이지에서 가독성이 향상됩니다

### 버그 수정

- **페이지 제목과 첫 섹션 사이 불필요한 여백 수정** — 페이지 제목 바로 아래 첫 번째 섹션 제목에 과도한 상단 여백이 표시되던 문제를 수정하여 콘텐츠가 더 자연스럽게 이어집니다
- **페이지 로딩 시 레이아웃 깜빡임 해소** — 디자인 시스템 41개 페이지에서 스타일이 뒤늦게 적용되며 콘텐츠가 밀리던 현상을 수정하여 더 부드러운 로딩 경험을 제공합니다
- **인터랙티브 컴포넌트 렌더링 오류 수정** — 35개 페이지에서 Tabs, Accordion, Steps, Note 내부 콘텐츠가 코드 블록으로 잘못 표시되던 문제를 수정했습니다
- **버튼 위계 페이지 렌더링 복구** — [버튼 위계](https://docs.sazo.kr/design-system/가이드/버튼-위계) 페이지에서 특수 문자로 인해 페이지가 표시되지 않던 문제를 수정했습니다
- **다이어그램 레이아웃 수정** — [버튼 위계](https://docs.sazo.kr/design-system/가이드/버튼-위계), [텍스트 계층](https://docs.sazo.kr/design-system/가이드/텍스트-계층), [카드 조합](https://docs.sazo.kr/design-system/레시피/카드-조합), [린터 스펙](https://docs.sazo.kr/design-system/콘텐츠/린터-스펙) 페이지에서 코드 블록이 프레임과 충돌하여 레이아웃이 깨지던 문제를 수정했습니다
- **콘텐츠 카드 아이콘 수정** — [디자인 시스템 개요](https://docs.sazo.kr/design-system/index) 페이지에서 콘텐츠 영역 카드의 아이콘이 표시되지 않던 문제를 수정했습니다
